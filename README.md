# Travelling Equipment Store

## Installation

### Clone Repository
git clone https://github.com/Muskan5103/Travelling-Equipment-Store.git

cd Travelling-Equipment-Store

### Create Virtual Environment
python -m venv venv

### Activate Environment

Windows:
venv\Scripts\activate

Linux/Mac:
source venv/bin/activate

### Install Dependencies
pip install -r requirements.txt

### Apply Migrations
python manage.py migrate



### Run Server
python manage.py runserver

Visit:
http://127.0.0.1:8000/
